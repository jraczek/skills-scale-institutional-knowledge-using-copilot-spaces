# OctoAcme - Role Accountability Matrix (RACI) Template

## Purpose
Use this template during initiation and planning to make ownership explicit for major decisions, deliverables, and approvals.

## How to use
1. Copy this table into your project charter or planning notes.
2. For each activity, assign exactly one Accountable role.
3. Confirm all Consulted and Informed roles before execution starts.
4. Revisit after scope changes, escalations, and major release plan updates.

## RACI Definitions
- Responsible: does the work.
- Accountable: owns the outcome and final decision.
- Consulted: provides required input before decisions.
- Informed: kept updated on status and outcomes.

## Minimum Quality Rules
- Exactly one Accountable per row.
- No blank ownership cells for required activities.
- Activities with compliance impact must include Security / Compliance as Consulted or Accountable.
- Activities with customer-facing impact must include Customer Support / Operations as Consulted or Informed.
